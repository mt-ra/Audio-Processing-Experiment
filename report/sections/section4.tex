\subsection{The Z-transform of a signal}
For a real time domain signal $x:\mathbb{Z}\to\mathbb{R}$, its $z$ transform is a function $X:\mathbb{C}\to\mathbb{C}$, where
$$X(z)=\sum_{n=-\infty}^{\infty}x[n]z^{-n}.$$
